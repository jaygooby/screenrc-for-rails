screenrc-for-rails by jaygooby

Fed up with starting server, console and mysql in your terminal? Put this in the root of your rails app and it'll do it all for you with a nice named window list to boot: "screen -c .screenrc" to execute it

Install

Copy to the root of your rails project, make sure the mysql call has the correct development database named
